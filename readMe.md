# Simple Calculator App

A simple calculator built using **HTML**, **CSS**, and **ES6 JavaScript**. The application performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an interactive user interface.

## Features

* Perform basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Clear calculator display
* Responsive design
* User-friendly interface
* Built with modern ES6 JavaScript syntax

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/calculator-app.git
```

2. Navigate to the project folder:

```bash
cd calculator-app
```

3. Open `index.html` in your browser.

No additional installations or dependencies are required.
